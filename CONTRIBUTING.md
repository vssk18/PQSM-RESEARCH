# Contributing to PQSM Research

Thank you for your interest in contributing!

## Development Setup
```bash
git clone https://github.com/vssk18/PQSM-RESEARCH.git
cd PQSM-RESEARCH
python -m venv venv
source venv/bin/activate
pip install -e ".[dev]"
pytest tests/ -v
```

## Important: Research Integrity

This repository supports a published research paper. When contributing:

**DO NOT change:**
- Experimental parameters (latencies, loss rates, etc.)
- Data generation logic
- Any reported numbers or results

**Welcome contributions:**
- Code quality improvements
- Documentation
- Additional tests
- Bug fixes
